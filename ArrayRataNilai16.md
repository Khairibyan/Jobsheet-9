Jawaban pertanyaan percobaan 3

jawaban:

1. Pada program ini, ditambahkan variabel baru jumlahLulus untuk menghitung jumlah mahasiswa yang lulus.
Program akan memeriksa setiap nilai dalam array, dan jika nilainya lebih besar dari 70, maka jumlahLulus akan bertambah satu.
Setelah perulangan selesai, program menampilkan jumlah mahasiswa yang lulus serta nilai rata-ratanya.

2. Pada modifikasi ini, jumlah mahasiswa ditentukan melalui input pengguna. Setelah itu, program meminta nilai setiap mahasiswa dan menyimpannya dalam array.
Program kemudian memisahkan nilai berdasarkan kriteria kelulusan, yaitu nilai lebih dari 70 sebagai lulus dan nilai 70 ke bawah sebagai tidak lulus.
Setelah semua data diproses, program menghitung rata-rata nilai untuk mahasiswa yang lulus dan rata-rata nilai untuk mahasiswa yang tidak lulus.
